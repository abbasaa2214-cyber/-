# حاسبة الراتب التقاعدي — مشروع أندرويد (Capacitor)

هذا مشروع Capacitor كامل وجاهز، يحوّل التطبيق إلى ملف APK حقيقي يُثبَّت بشكل تقليدي على أي جهاز أندرويد.

## محتوى المشروع

- `www/` — كود التطبيق (HTML/JS) نفسه المستخدم بنسخة الويب
- `capacitor.config.json` — إعدادات التطبيق (الاسم، المعرّف، الأيقونة)
- `package.json` — قائمة الحزم المطلوبة
- `resources/` — صورة الأيقونة الأساسية (512×512) لتوليد كل أيقونات أندرويد منها

## الطريقة الموصى بها: استخدام منصة بناء سحابية (بدون تنصيب أي شيء)

أسهل طريقة تبني فيها APK من هذا المشروع بدون تنصيب Android Studio على جهازك هي استخدام منصة CI سحابية. الخطوات العامة لأي منصة من هذي:

### الخيار أ: Appflow (من شركة Capacitor/Ionic نفسها)
1. سجّل حساب مجاني على https://ionic.io/appflow
2. ارفع هذا المشروع على GitHub (مستودع خاص أو عام)
3. اربط المستودع بـ Appflow من لوحة التحكم
4. اضغط "Build" واختر Android → يبني لك APK جاهز للتحميل مباشرة

### الخيار ب: GitHub Actions (مجاني تماماً، يحتاج إعداد ملف بسيط)
1. ارفع المشروع على GitHub
2. أضف ملف `.github/workflows/build.yml` يستخدم إجراءات جاهزة لبناء Capacitor (يوجد قوالب جاهزة بحثها على GitHub باسم "capacitor android build action")
3. كل دفعة (push) تبني APK تلقائياً تجده بقسم Actions → Artifacts

### الخيار ج: تنصيب محلي (إذا عندك جهاز قوي ووقت)
```
npm install
npx cap add android
npx cap sync
npx cap open android
```
يفتح Android Studio (يحتاج تنصيبه مسبقاً)، ومنه تضغط Build → Build APK.

## بعد الحصول على APK

ترسله لنفسك أو لأي شخص عبر واتساب/تيليجرام، يفتحه على أندرويد، يفعّل "تثبيت من مصادر غير معروفة" إذا طُلب منه، ويثبته بشكل طبيعي.

## ملاحظة عن شرح الذكاء الاصطناعي

نفس الميزة بنسخة الويب (تحتاج مفتاح Anthropic API شخصي يُحفظ محلياً على الجهاز عبر localStorage). باقي الحسابات تعمل بدون أي مفتاح أو إنترنت.
